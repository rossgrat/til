# Hypervisors and Virtualization

Virtualization is the process of creating a software based representation of physical hardware.

There are two types of virtualization. The first is bare-metal, where a hypervisor runs directly on the metal with no host OS underneath. The second type is hosted, where there is a host OS underneath the hypervisor.

A hypervisor enables bare-metal virtualization. A hypervisor runs in place of a regular OS, and allows multiple virtualized host OSes to share the underlying physical compute.
